E-commerce

El proyecto consiste en una aplicación que permite visualizar un listado de productos tecnológicos con una información detallada de cada uno al darle click en este. Tiene un sistema de autenticación con outh0, register y login con outh0. Cuenta con roles para user y admin. El usuario, puede hacer compras, editar su perfil y suscribirse al newsletter de la página, agregar a favoritos y comprar productos. El panel del admin, puede crear o editar un producto, ver lar ordenes de los productos, ver los comentarios de todos los productos y puede todos los usuarios, pudiéndole darle el rol de admin

Las herramientas utilizadas fueron: JavaScript, HTML, CSS, NodeJS, Express, Sequelize, PostgreSQL, React, Redux, cloudinary.
